無痕風颱工作台 (Hong-Thai Incognito Reader) 隱私權政策

最後更新日期：2025年7月30日

感謝您使用「無痕風颱工作台」（以下簡稱「本擴充功能」）。我們極度重視您的隱私權。本隱私權政策旨在說明本擴充功能如何處理您的資料。

核心原則：您的資料，由您掌控

本擴充功能的設計核心是**完全在您的瀏覽器本機端運作**。我們**沒有**任何後端伺服器，也**不會**收集、儲存、或傳輸任何您的個人可識別資訊或您處理的內容。

資料處理說明

1. Google Gemini API 金鑰

  • 本擴充功能需要您提供自己的 Google Gemini API 金鑰才能運作。
  • 您的金鑰僅會透過 Chrome 的標準儲存空間 (chrome.storage.sync) **加密儲存於您自己的瀏覽器中**。此金鑰只會被用來向 Google Gemini 的官方 API 端點發送請求。
  • 我們（開發者）絕不會存取、看見或儲存您的 API 金鑰。

2. 您處理的內容（文字、圖片、語音）

  • 一般功能：當您使用「翻譯當前網頁」、「文字/截圖工作台」或「無痕-語音輸入筆記」功能時，為了在不同分頁間傳遞資料，處理中的文字內容可能會**暫時**儲存於您本機的 chrome.storage.local 中。這些暫存資料會在您關閉對應的功能分頁後被清除。
  • 無痕-Gemini 2.5 對話框 (最高隱私模式)：當您使用此功能時，所有的對話歷史（包含文字與圖片）**只會儲存在您電腦的記憶體 (RAM) 中**。這些資料**不會**寫入任何硬碟儲存區 (包含 chrome.storage)。**一旦您關閉該分頁，所有對話紀錄將立即永久消失，無法復原。**
  • 資料流向：在所有情況下，您的內容都只會**在您的瀏覽器與 Google 的伺服器之間直接傳輸**，不會經過或儲存在任何由我們（開發者）控制的伺服器上。
  • Google 的政策：根據 Google 的 API 政策，他們可能保留 API 請求的內容（不含個人資訊）最長 30 天，僅用於服務維護與安全稽核，不會用於模型訓練。

3. （選用）雲端日誌功能

  • 本擴充功能提供一個**選用**的日誌功能，讓您可以將操作紀錄傳送到**您自己指定**的雲端端點（例如您自己的 Google Cloud Function）以供研究分析。
  • 此功能預設為關閉。只有在您明確於設定頁面中填入您自己的端點網址與金鑰時，才會啟用。
  • 所有日誌資料的流向完全由您自己控制。

權限使用說明

  • activeTab: 僅在您主動點擊擴充功能圖示時，臨時存取您當前的分頁，以便進行網頁擷取。
  • storage: 用於在本機安全地儲存您的 API 金鑰與設定。
  • debugger: 僅在您使用「翻譯當前網頁」功能時，臨時啟用以擷取完整的網頁畫面，擷取後立即關閉。

政策變更

我們可能會更新本隱私權政策。任何變更都將發布在此頁面上。我們鼓勵您定期查看本政策以了解最新資訊。
